# 程序重命名说明

## 变更内容

程序名称从 `svn-reviewer` 更改为 `svn-ai-reviewer`，以更好地体现程序使用 AI 进行代码审核的特性。

## 修改清单

### 1. Go 模块名称
- **文件**: `go.mod`
- **修改**: `module svn-code-reviewer` → `module svn-ai-reviewer`

### 2. 导入路径
修改了所有 Go 文件中的导入路径：

**主程序**:
- `main.go`

**命令行**:
- `cmd/root.go`
- `cmd/review.go`
- `cmd/online.go`

**GUI 服务器**:
- `gui/server.go`

**内部包**:
- `internal/ai/client.go`
- `internal/ai/openai.go`
- `internal/report/html.go`

### 3. 可执行文件名
- **旧名称**: `svn-reviewer.exe`
- **新名称**: `svn-ai-reviewer.exe`

### 4. .gitignore
更新了忽略规则：
```
svn-ai-reviewer
svn-ai-reviewer.exe
```

### 5. 文档更新
批量更新了所有 Markdown 文档中的程序名称引用。

## 使用方式

### 构建程序

```bash
go build -o svn-ai-reviewer.exe
```

### 运行程序

**GUI 模式**:
```bash
# Windows
svn-ai-reviewer.exe

# Linux/Mac
./svn-ai-reviewer
```

**CLI 模式**:
```bash
# 本地审核
svn-ai-reviewer.exe review

# 在线审核
svn-ai-reviewer.exe review online --url <svn-url>
```

## 迁移步骤

如果您已经在使用旧版本，迁移非常简单：

### 方式一：直接替换（推荐）

1. 删除旧的可执行文件
   ```bash
   rm svn-reviewer.exe  # 或手动删除
   ```

2. 构建新的可执行文件
   ```bash
   go build -o svn-ai-reviewer.exe
   ```

3. 运行新程序
   ```bash
   svn-ai-reviewer.exe
   ```

### 方式二：保留旧版本

如果您想保留旧版本进行对比：

1. 重命名旧文件
   ```bash
   mv svn-reviewer.exe svn-reviewer-old.exe
   ```

2. 构建新版本
   ```bash
   go build -o svn-ai-reviewer.exe
   ```

3. 两个版本可以共存

## 为什么重命名？

### 1. 更准确的名称
- **旧名称**: `svn-reviewer` - 只表明是 SVN 审核工具
- **新名称**: `svn-ai-reviewer` - 明确表明使用 AI 技术

### 2. 更好的识别度
- 在文件列表中更容易识别
- 在进程列表中更容易区分
- 在文档中更容易理解功能

### 3. 更符合项目定位
- 核心特性是 AI 驱动的代码审核
- 与其他 SVN 工具区分开来
- 突出技术优势

## 注意事项

1. **首次运行**: 如果您使用的是新名称，首次运行时会自动加载现有配置
2. **快捷方式**: 如果您创建了桌面快捷方式，需要更新指向新的可执行文件
3. **脚本更新**: 如果您在脚本中使用了程序名称，需要更新为新名称
4. **文档引用**: 所有文档已自动更新，无需手动修改
